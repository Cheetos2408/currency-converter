# Currency Converter

## Описание проекта
Currency Converter — это веб-приложение для быстрого и удобного перевода валют в реальном времени. Пользователь может выбрать валюты, ввести сумму с помощью цифровой клавиатуры и сразу увидеть результат конвертации.

Проект ориентирован на мобильный интерфейс и простой пользовательский опыт без лишних элементов.



## Функциональность
- Перевод валют в реальном времени  
- Выбор валют из выпадающих списков  
- Автоматическая загрузка актуальных курсов  
- Отображение флагов стран  
- Кастомная цифровая клавиатура  
- Адаптация под мобильные устройства  


## Используемые технологии
- HTML — структура приложения  
- CSS — дизайн и адаптивная верстка  
- JavaScript — логика приложения, работа с DOM и API  
- ExchangeRate API — получение курсов валют  


## Запуск проекта
1. Скачайте проект или склонируйте репозиторий.  
2. Откройте файл `index.html` двойным кликом в браузере.  
3. Выберите валюты, введите сумму и получите результат конвертации.  

Никакой дополнительной настройки не требуется.

currency-converter/

## Работа с API
Приложение использует **ExchangeRate API** для получения актуальных курсов валют.

- Запрос выполняется через `fetch`
- Курсы обновляются при смене валют
- API не требует ключа доступа

Источник API:  
https://api.exchangerate-api.com



## Архитектура проекта
Проект построен как клиентское веб-приложение без сервера:

- HTML отвечает за структуру интерфейса  
- CSS отвечает за визуальный стиль  
- JavaScript управляет логикой, DOM и API-запросами  

Все вычисления происходят в браузере пользователя.



## Используемые ресурсы
- ExchangeRate API — https://api.exchangerate-api.com  
- Флаги стран — https://flagcdn.com
